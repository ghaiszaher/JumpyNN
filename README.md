# JumpyNN

This project is a trial to train a neural network to solve [Facebook's Jumpy Jumpy Game](https://www.facebook.com/instantgames/198982457542294)

## Dependencies

Make sure that the following are installed:
* Python3
* Keras
* Tensorflow
* Opencv2 & Numpy
* pynput
* pyautogui
* Pillow
